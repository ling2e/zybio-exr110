# POCT-EXR110 Series-Manuals-How to clear the system clock error of EXR110.pdf

How to clear the “system clock” error of EXR110

1. Error picture

2. Prepare a battery. Format: CR1220 3V

3. Remove the four screws by Hex key

4. Install this battery on the main control board

5. Install the four screws back, and power on, login the service account, user name: service Password: mar@zybek 6. Enter Management> Settings> System settings> System time, set system time and restart the instrument, the error will be disappeared.
