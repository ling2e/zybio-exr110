How to upgrade the EXR110 software

1. Check the instrument current software version User name: service Password: mar@zybek Enter Management> Service> Version, if the version is old, then you can upgrade.

2. Ask for the software upgrade package from Zybio service engineer, put the upgrade package “update\_Q3.tar.gz” in the root path of pen drive(USB disk), don’t unzip this file.

3. Connect USB disk to machine

4. Click “Upgrade”

5. Then wait for a moment, the instrument will enter update interface as below, click “Update”

6. The upgrade process will take approximately one minute. Click “OK” after finished, then power off the instrument and restart the instrument.

7. Then login the service account, check the software version again.
